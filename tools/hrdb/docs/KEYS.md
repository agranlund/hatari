Normal focus

- Ctrl+R - Start/stop
- S      - Step (execute next instruction)
- N      - Next (step through bsr/jsr)
- Alt+B  - Add breakpoint (dialog)

- Alt+D - Focus Disassembly View
- Alt+M - Focus Memory View
- Alt+G - Focus Graphics Inspector

Disassembly focus:

- Ctrl+B - Toggle breakpoint
- F3 Run to cursor
